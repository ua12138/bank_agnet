# 核心目标
用户要看到的不是：
- 下一跳：Workflow
- 输入样例：task.payload_json
- 输出样例：DONE

用户要看到的是：
- 到底是哪一行代码跳过去的
- 跳过去时带着什么字段
- 对方在哪个函数接住
- 接住后状态对象变成什么样
- 返回后怎么继续往下走

## 2. 入口与主链路位置
## 3. 全局链路图

- 代码片段
- 输入样例
- 传递参数/状态
- 接收位置
- 状态变化
- 返回后继续
